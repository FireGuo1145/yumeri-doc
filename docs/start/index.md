## 安装

最方便的项目初始化方式是使用项目模板：

::: code-group

```bash [npm]
npm init yumeri
```

```bash [yarn]
yarn create yumeri
```

当然，如果不想使用项目模板，也可手动安装yumeri依赖：

:::

::: code-group

```bash [npm]
npm init
npm install yumeri
```

```bash [yarn]
yarn add yumeri
```

:::
